Degree Planning App

To gain experience in Java and in mobile app development, I have completed an app that helps students plan their degree. It is currently available on the app store here. Planning what courses to take and when to take them is a headache that every student knows, and this is a tool to help alleviate some of that pain! The current version only supports the most basic class planning, but you can be certain that I am working on an update to offer the best product possible to my peers. It will have features that enable yearly class planning, GPA calculating, and semester planning with ease. The current version uses java as the source language (for more about my escapades with Kotlin, see the current projects section below). I have implemented many of the new android architecture components, such as the new Room Persistence Library and ViewModel class.

Robot Project 2018

In ENGR120, a team of three was required to build a working model for an autonomous undersea cable-laying robot (shown above). The objective was for the robot to take a cable from a spool and take it to a beacon, where it had to be attached. In my group, I was the team leader and main programmer. The entire project used VEX microcontrollers, and was coded in a variant of C. Our project did so well that we received an A for the project and were invited to the Uvic Robotics competition, which we did compete in. The whole project was designed and developed from January to March 2018. This project inspired me to start a robot project of my own, which I am currently working on with a colleague. I look forward to the future experience that I hope to gain in this field.

To work more with robotics, embedded systems, microprocessors and microcontrollers, I am designing, programming, and building an autonomous arduino-based robot. However, we have made final decisions about using an Arduino Uno The idea for it was inspired by my grandfather, who due to his age, is less and less willing to get up from his chair and get his nightly cola.
